# NJ Construction Website

A modern, professional website for NJ Construction - a leading construction company offering residential, commercial, interior design, and renovation services.

## Features

- **Responsive Design**: Fully responsive and mobile-friendly layout
- **Modern UI**: Clean and professional design with gold and silver color scheme from the logo
- **Multi-page Layout**: 
  - Home (index.html) - Hero section with services overview
  - About (about.html) - Company mission, vision, and values
  - Services (services.html) - Detailed service information
  - Portfolio (portfolio.html) - Project gallery with filtering
  - Team (team.html) - Team member profiles
  - Testimonials (testimonials.html) - Client reviews and ratings
  - Contact (contact.html) - Contact form and information

- **Interactive Elements**:
  - Mobile hamburger menu
  - Smooth scrolling navigation
  - Portfolio filtering
  - Contact form validation
  - Hover effects and animations

## Getting Started

1. Clone or download the repository
2. Replace the logo placeholder (assets/logo.png) with your actual logo
3. Update contact information in footer and contact page
4. Customize content and images as needed
5. Deploy to your web hosting

## Customization

### Update Contact Information
Update dummy contact details in:
- Footer sections (all pages)
- Contact page (contact.html)

### Update Logo
Place your logo file at `assets/logo.png`

### Modify Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-gold: #D4AF37;
    --secondary-gold: #C9A961;
    --silver: #C0C0C0;
    --dark-bg: #1a1a1a;
    /* ... */
}
```

### Add Real Portfolio Images
Replace the gradient placeholders in portfolio.html with actual project images
